##  Функционал
- **Поиск городов:** Мгновенный поиск погоды по названию.
- **Геолокация:** Автоматическое определение и добавление текущего местоположения при первом запуске.
- **Избранное:** Сохранение списка городов и их удаление в `localStorage` (данные не теряются при перезагрузке).
- **Погода в течение 12 часов:** Отображение изменения погоды в течение 12 часов (+ показатель влажности).
- **Погода на 6 дней:** Прогноз погоды на 6 дней вперед.
- **Детали:** Отображение таких деталей, как влажность, давление, "ощущается как" и т.д.
- **Единицы измерения:** Переключение между Цельсием (°C) и Фаренгейтом (°F).
- **Адаптивность:** Корректное отображение на мобильных устройствах и десктопах.

## Стек технологий
- **HTML5** (семантическая верстка)
- **CSS3** (Flexbox, CSS Grid, переменные, адаптивный дизайн)
- **SASS/SCSS** (для удобства вёрстки)
- **Vanilla JavaScript (ES6+)** (DOM-манипуляции, Fetch API, Async/Await, LocalStorage)
- **API:** 
  - [Open-Meteo API](https://open-meteo.com/) (погода и прогноз)
  - [Nominatim (OpenStreetMap)](https://nominatim.org/) (обратное геокодирование для определения города по координатам)
